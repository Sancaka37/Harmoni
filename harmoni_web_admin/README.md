# Harmoni Web Admin

Versi web statis dari desain admin `HARMONI — Pusat Oleh-oleh`.

## Isi Paket

- `index.html` — halaman utama web admin.
- `assets/css/style.css` — seluruh styling responsif.
- `assets/js/app.js` — navigasi halaman, filter, pencarian, modal, export CSV, dan interaksi demo.
- `assets/images/` — aset gambar produk dari file desain yang diunggah.

## Cara Menjalankan

1. Ekstrak file ZIP.
2. Buka `index.html` langsung melalui browser.
3. Tidak perlu instalasi server, database, atau library tambahan.

## Fitur Demo

- Dashboard analytics.
- Halaman Orders dengan pencarian, filter status, filter pembayaran, checkbox, dan export CSV.
- Halaman Products dengan filter kategori dan modal tambah produk.
- Halaman Customers dengan pencarian dan filter tipe pelanggan.
- Desain responsif untuk desktop, tablet, dan mobile.

## Catatan

Web ini masih berbasis data dummy di file `assets/js/app.js`. Untuk dijadikan aplikasi produksi, data dapat dihubungkan ke backend seperti Laravel, Express, Nuxt API, Firebase, Supabase, atau database internal.
